1.What is DOM and what is the purpose of DOM?

DOM refers to Document Object Model. DOM is the programming interface for a HTML document.
The purpose of DOM is to interrupt inside the html document that means how to use or change the property of the html document by using javascript.In the DOM, all the HTML elements are defined as objects. And it performs all the task from inside the javascript.

3. What is event bubble?

If there's a event that is ongoing or clicked and there's also events in it's parent node than these events are also be performed or clicked.
If we want to perform only target event not these events that are in the parent node than we should use stopPropagation .

4. What is a callback function and why will you use it?

The function that is called itself is called callback function.
we use this callback function to reduce the redundancy of the codes. sometimes same type of task can be done again and again , in this situation function are very helpful for us..again in the function, there also do the same task again and again.than we call the same function inside the function.And this type of function is called callback function.
